Projetos

Projeto 1

Projeto 1: Desenvolvimento de Site de Viagens Este projeto envolveu o desenvolvimento de um site inovador de viagens chamado "TravelExplorer". O site oferece aos usuários a capacidade de explorar destinos de viagem emocionantes em todo o mundo, reservar hotéis, verificar preços de voos e planejar itinerários personalizados. Implementamos recursos avançados, como uma calculadora de orçamento de viagem e uma seção de avaliações de viajantes para garantir uma experiência completa. Este projeto exigiu habilidades de desenvolvimento web, integração de APIs de terceiros e uma interface de usuário altamente responsiva.

Projeto 2

Projeto 2: Campanha Publicitária para Lançamento de Produto O projeto "Lançamento de ProdutoX" envolveu a criação de uma campanha publicitária abrangente para o lançamento de um produto revolucionário. Fui responsável por toda a estratégia de marketing, desde a criação de anúncios atraentes para redes sociais até a produção de vídeos promocionais. A campanha incluiu também a criação de um site de destino personalizado para o produto, onde os clientes puderam obter mais informações e fazer compras. O resultado foi um lançamento de produto bem-sucedido que gerou um aumento significativo nas vendas e reconhecimento da marca.
